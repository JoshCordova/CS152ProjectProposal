Project Title: Music Classifier


By using the dataset found on http://millionsongdataset.com/, the plan is to deploy a neural network
to be used in an application that will classify music. Since this dataset does not contain genre tags,
We will not be able to use supervised learning techniques, so this will be done using clustering. To use the application,
users will upload a song, and the features will be looked up using the Spotify (formerly Echo Nest) API.

If the clusters are distinct enough, and seem to represent established genres, labels can be added post-clustering to better inform the user about the specific genre of music.
If the clusters are not distinct after enough hyperparameter tuning, we can use soft/fuzzy classification to try to come up with a recommender system, and compare it to Spotify's recommendations.

Specific goals: 
 
  • Run cluster analysis on the dataset
  • Tune hyperparameters (distance metric will likely be the main focus) to get distinct clusters
  • Verify that these clusters represent genres
  • Add genre tags and train a classification model
